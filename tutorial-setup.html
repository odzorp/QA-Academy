Environment Setup

Configure your machine for LoadMagic.AI testing

Duration: 20 minutes | Level: Beginner | Framework: LoadMagic.AI

What You'll Learn

  • Install and configure JMeter plugin

Prerequisites

Before You Begin

  • A computer with internet access
  • Modern web browser (Chrome, Firefox, or Edge)
  • Email address for account creation
  • For Desktop: Java 11+ installed

  3. Enter your email address and create a password
  4. Check your email and verify your account

Step 2: JMeter Plugin (Desktop)

For teams that prefer local testing or need offline capabilities, install the JMeter plugin.

Windows Installation

1 Install Java (if not already installed)
# Check if Java is installed java -version # If not installed, use Chocolatey choco install openjdk17
2 Download JMeter
  1. Download from jmeter.apache.org
  2. Extract to C:\Program Files\Apache\JMeter
  3. Add JMeter bin to PATH
3 Download LoadMagic Plugin
  1. Visit loadmagic.ai/more
  2. Download LoadMagic-Plugin-*.jar
  3. Copy to JMETER_HOME\lib\ext\
4 Launch JMeter
# Set API key and start JMeter $env:LOADMAGIC_API_KEY="your_api_key" jmeter.bat

macOS Installation

1 Install Java and JMeter
# Using Homebrew brew install jmeter # Or download from apache.org
2 Install LoadMagic Plugin
# Copy plugin to JMeter lib/ext cp LoadMagic-Plugin-*.jar /usr/local/opt/jmeter/libexec/lib/ext/
3 Launch JMeter
# Set API key and start export LOADMAGIC_API_KEY="your_api_key" open /usr/local/opt/jmeter/bin/jmeter

Linux Installation

1 Install Dependencies
# Ubuntu/Debian sudo apt update sudo apt install openjdk-17-jre # Download JMeter wget https://archive.apache.org/dist/jmeter/binaries/apache-jmeter-5.7.tgz tar -xzf apache-jmeter-5.7.tgz
2 Install LoadMagic Plugin
# Copy plugin cp LoadMagic-Plugin-*.jar apache-jmeter-5.7/lib/ext/
3 Launch JMeter
# Set API key and start export LOADMAGIC_API_KEY="your_api_key" ./apache-jmeter-5.7/bin/jmeter

Step 3: CLI Setup

For automation and CI/CD integration, set up the LoadMagic CLI.

1 Install via npm
# Install globally npm install -g @loadmagic/cli # Verify installation loadmagic --version
2 Authenticate
# Login with your account loadmagic login # Or use API key loadmagic config set api-key YOUR_API_KEY
üí° Pro Tip

Store your API key securely using environment variables instead of hardcoding in scripts!

Step 4: Environment Variables

Configure your environment for optimal performance and security.

Cloud Configuration

# LoadMagic Cloud Configuration export LOADMAGIC_API_URL="https://api.loadmagic.ai" export LOADMAGIC_REGION="us-east-1" export LOADMAGIC_AUTH_TYPE="api_key" export LOADMAGIC_API_KEY="your_api_key_here"

Windows (PowerShell)

# Set environment variables $env:LOADMAGIC_API_KEY = "your_api_key_here" $env:LOADMAGIC_REGION = "us-east-1"
