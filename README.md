# LoadMagic-AI — Personal Portfolio & Tutorials

This is the personal portfolio and tutorial site for LoadMagic.AI, maintained by odzorp (Philip). It contains a short personal profile, project showcase, and step-by-step tutorials aimed at helping users become proficient with LoadMagic.AI.

## About

This repository hosts a static website that serves as my personal portfolio and a learning resource for LoadMagic.AI. The site includes a profile, contact details, resources, and tutorials (including a quickstart) that walk through using LoadMagic.AI effectively.

## What you'll find here

- `index.html` — Home / introduction
- `about.html` — Personal profile and background
- `contact.html` — Contact information and links
- `resources.html` — Helpful links and references
- `tutorials.html` & `tutorial-quickstart.html` — Beginner → advanced tutorials for LoadMagic.AI
- `assets/` — Images and media used on the site
- `css/` — Styles (`styles.css`)
- `js/` — Client-side scripts (`main.js`)

## Local Preview

Preview the site locally with a static server (Python example):

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Or use Node-based servers such as `live-server` or `http-server`.

## Deploy

This site is suitable for GitHub Pages or any static host. To deploy to GitHub Pages, push the repository to your GitHub account and enable Pages in the repository settings.
