How They Work Together

The five agents collaborate in a pipeline to transform your HAR file into production-ready scripts:

# Example: Agent Pipeline Execution
pipeline = LoadMagicPipeline()

# Step 1: HAR Processing
har_data = pipeline.process_har(har_file)

# Step 2: George generates JMeter/Locust script
script = pipeline.george.generate(har_data)

# Step 3: Rupert extracts dynamic values
correlations = pipeline.rupert.extract(har_data)

# Step 4: Suzy documents and optimizes
script = pipeline.suzy.optimize(script)

# Step 5: Carrie handles session management
script = pipeline.carrie.correlate(script, correlations)

# Step 6: Quinn validates and heals
validated_script = pipeline.quinn.validate(script)
