AI Agents Deep Dive

Meet your AI testing team

Duration: 35 minutes | Level: Intermediate | Framework: LoadMagic.AI

What You'll Learn

  • Understanding each AI agent's role
  • How agents collaborate to generate test scripts
  • When to leverage each agent's strengths
  • Troubleshooting agent-specific issues

1. George - JMeter Genie

🤖
George
JMeter Genie - The Script Architect

George is your JMeter expert. He transforms HAR recordings into production-ready JMeter test plans with optimal configurations.

🎯 JMeter Script Generation ⚙️ Test Plan Optimization 🔧 Configuration Management 📊 Performance Best Practices

What George Does

  • Parses HAR files and identifies HTTP requests
  • Generates JMeter test plans with proper structure
  • Configures thread groups, samplers, and assertions
  • Optimizes scripts for performance and maintainability

Example Output

from locust import HttpUser, task, between

class JMeterGeneratedUser(HttpUser):
    wait_time = between(1, 3)
    
    def on_start(self):
        self.client.post("/api/login", json={
            "username": "testuser",
            "password": "testpass"
        })

2. Rupert - Regex Expert

📝
Rupert
Regex Expert - The Pattern Master

Rupert specializes in identifying and extracting dynamic values using advanced regex patterns.

🔍 Pattern Matching 📋 Dynamic Value Extraction 🎯 Data Correlation 💡 Regex Optimization

What Rupert Does

  • Identifies dynamic values in responses (tokens, IDs, timestamps)
  • Creates regex patterns to extract these values
  • Builds parameterization rules for reusable scripts
  • Validates pattern accuracy across multiple responses

3. Suzy - Script Scholar

📜
Suzy
Script Scholar - The Python Expert

Suzy is your Locust specialist, creating clean, well-documented Python scripts.

🐍 Locust Scripts 📖 Code Documentation ✨ Code Optimization 🐛 Bug Prevention

4. Carrie - Correlator

🔗
Carrie
Correlator - The Connection Expert

Carrie handles all aspects of session management and authentication correlation.

🍪 Session Management 🔐 Authentication Handling 🔄 Request Chaining 📊 State Management

5. Quinn - QA Assessor

✅
Quinn
QA Assessor - The Quality Guardian

Quinn ensures test quality, manages the self-healing pipeline, and validates results.

🛡️ Quality Validation 🔄 Self-Healing 📈 Results Analysis ⚠️ Error Detection

How They Work Together

The five agents collaborate in a pipeline to transform your HAR file into production-ready scripts:

# Example: Agent Pipeline Execution
pipeline = LoadMagicPipeline()

# Step 1: HAR Processing
har_data = pipeline.process_har(har_file)

# Step 2: George generates JMeter/Locust script
script = pipeline.george.generate(har_data)

# Step 3: Rupert extracts dynamic values
correlations = pipeline.rupert.extract(har_data)

# Step 4: Suzy documents and optimizes
script = pipeline.suzy.optimize(script)

# Step 5: Carrie handles session management
script = pipeline.carrie.correlate(script, correlations)

# Step 6: Quinn validates and heals
validated_script = pipeline.quinn.validate(script)
