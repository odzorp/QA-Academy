Quick Start Guide

Create your first load test in 15 minutes

Duration: 15 minutes | Level: Beginner | Framework: LoadMagic.AI + Locust

What You'll Learn

  • Record a HAR file from your browser
  • Upload and process with LoadMagic.AI
  • Generate a Locust test script
  • Run your first load test
  • Analyze results

Prerequisites

  • LoadMagic.AI account (free tier works)
  • Chrome or Firefox browser
  • A target API or website to test

Step 1: Record a HAR File

HAR (HTTP Archive) files capture all HTTP traffic from your browser. Here's how to create one:

  1. Open Chrome and press F12 to open DevTools
  2. Click the Network tab
  3. Check the Preserve log option
  4. Perform your test actions (login, browse, etc.)
  5. Right-click in the Network panel → Save all as HAR with content
Tip: Record a realistic user flow - login → browse → perform action → logout

Step 2: Upload to LoadMagic.AI

  1. Go to studio.loadmagic.ai
  2. Create a new project or open existing
  3. Click Drop/Browse HAR and select your .har file
  4. Wait for processing to complete

Step 3: Generate Test Script

LoadMagic.AI automatically:

  • Parses the HAR file
  • Identifies endpoints and methods
  • Extracts authentication patterns
  • Creates a Locust Python script
"""
Generated by LoadMagic.AI
"""
from locust import HttpUser, task, between

class APITestUser(HttpUser):
    wait_time = between(1, 3)
    
    def on_start(self):
        response = self.client.post("/api/login", json={
            "username": "test",
            "password": "test123"
        })
        self.token = response.json().get("token")
    
    @task(3)
    def get_users(self):
        self.client.get("/api/users",
            headers={"Authorization": f"Bearer {self.token}"})
    
    @task(2)
    def get_profile(self):
        self.client.get("/api/profile",
            headers={"Authorization": f"Bearer {self.token}"})

Step 4: Run Your Test

Configure load parameters:

  • Users: 100 (start small)
  • Spawn Rate: 10 users/second
  • Duration: 5 minutes

Click Run and watch the real-time results!

Step 5: Analyze Results

Key metrics to watch:

Metric Target Alert If
Success Rate > 99% < 95%
Avg Response Time < 500ms > 1s
p95 Response Time < 1s > 2s
Error Rate < 1% > 5%
